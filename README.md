# mutual-fund-analytics-project
Mutual Fund Analytics project focused on data ingestion, NAV data collection from MFAPI, data validation, quality checks, and preparation for analytics dashboards.

## Tech Stack

Python

Pandas

SQLite

Plotly

Seaborn

Jupyter

## Project Structure

data/

notebooks/

scripts/

reports/

sql/

## How to Run

1. Clone repository

2. Install requirements

pip install -r requirements.txt

3. Run notebooks

4. View dashboard
